# Описание игры
* Игра представляет из себя 2D шутер, вид сверху.
* Игроки ходят по полю, вокруг спавнятся враги.
* Игрок стреляет во врагов, враги атакуют ближайшего игрока.
* игроки могут так же попадать в других игроков, каждое попадание наносит столько же урона как враг


# Управление
* Передвижение игрока проиходит при нажатии на клавиатуре кнопок w, s, a, d
* Стрельба при нажатии левой кнопки мыши по направлению от игрока в сторону курсора мыши

# Цветовая схема
* Игрок видит своего персонажа зеленым
* Другие игроки, фиолетовый
* Враги, красные

# Авторизация
* При подключении игрок получает ID сессии
* После чего взаимодействует с сервером по этому ID
* Если у клиента еще нет ID, сервер генерирует новый ID и выдает его клиенту
* Предусмотреть отключение от игры и деспавн персоонажей, при отключении клиента

# Начисление очков
* При попадании по игроку 20 очков
* При попадании в монстра 10 очков
